# Vision: Audio Extractor

## Goal

Make it effortless for anyone to extract audio from large video files — without needing to know command-line tools or codec details.

## Users

- **Content consumer**: A non-technical person who has video files and wants to pull out audio segments (e.g., a memorable quote, a song, a podcast episode recorded as video).

## Core Features

- **Video preview**: Scrub through large video files to find the exact section to extract.
- **Time-range extraction**: Select a start and end time to extract a specific audio clip.
- **Whole-audio extraction**: Extract the complete audio track with a single click.
- **Format choice**: Export audio as AAC, MP3, or OGG without needing to understand codecs.
- **Multiple clips**: Extract several segments from the same video in one session.

## Key Workflows

1. Open a video file using the file picker.
2. Preview the video to locate the desired section.
3. Set start/end times (or choose "extract whole audio").
4. Pick an output format and extract.
5. Repeat for additional clips from the same video if needed.

## Success Criteria

- Extraction works reliably on large video files without crashes or hangs.
- A non-technical user can extract audio without any guidance or documentation.
- The app provides clear progress feedback and handles errors gracefully.
